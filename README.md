# правила создания вопросов и ответов

Структура каждого вопроса:
1) Сам Вопрос
2) Мотивация вопроса. Почему он важен.
3) Тема вопроса. 
4) Ответ(ы) .
5) Используемые ресурсы. (Например: Ссылки на материалы. Номер страницы конкретной книги. И т.п.)
